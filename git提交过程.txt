1、创建文件夹，进入文件夹里面
2、git init
3、git remote add origin git@github.com:milko1991/vuedemo.git
	如果报错：先 $ git remote rm origin
4、git pull origin master
5、touch 文件名.xx
6、git add .
7、git commit -m ‘随意’
8、git push -u origin master


添加最近更新文件
1、git add -u
2、git commit -m ‘随意’
3、git push -u origin master

添加一个文件
1、进入要添加的文件目录里
2、git add 文件名.xx
3、git commit -m ‘随意’
4、git push -u origin master
